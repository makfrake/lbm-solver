# Lattice Boltzmann Python Solver

Straight forward implementation of the LBM to simulate fluid flow around simple 2D geometries like circles or squares.
The solver is based on the work done by **Philip Mocz** and can be found at the [following link](https://medium.com/swlh/create-your-own-lattice-boltzmann-simulation-with-python-8759e8b53b1c)
